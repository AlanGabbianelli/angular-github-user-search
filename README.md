Try it online
-------------
Visit [Github User Search](https://angular-github.herokuapp.com/)

Try it locally
--------------
```
$ git clone git@github.com:AlanGabbianelli/angular-github-user-search.git
$ cd angular-github
$ npm install
$ open index.html
```
